## Image Specifications

### General Guidelines:
- **Format:** JPG for photos, PNG for logos/transparent images
- **Quality:** High resolution, professional photography
- **Compression:** Optimize for web (use tools like TinyPNG)
- **Style:** Clean, modern, medical/professional aesthetic
- **Color:** Natural lighting, professional color grading

### Specific Size Requirements:

| Image Type | Minimum Size | Aspect Ratio | Use Case |
|-----------|--------------|--------------|----------|
| Hero Images | 1920x1080px | 16:9 | Main hero sections |
| Team Photos | 800x800px | 1:1 | Team member cards |
| Treatment Cards | 600x400px | 3:2 | Treatment/service cards |
| Clinic Photos | 1920x1080px | 16:9 | Carousel/showcase |
| Section Images | 1200x800px | 3:2 | About/services sections |

## Where to Add New Images

1. Place all new images in the `Images/` folder
2. Use descriptive, lowercase filenames with hyphens (e.g., `clinic-reception-area.jpg`)
3. Update the HTML file paths to match your new filenames
4. Test images load correctly on all devices
